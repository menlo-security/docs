Supported Documents and Files

Introduction

This page provides a list of all currently supported document/file formats. See the following sections for details:

  • 2-0supported-docs-docs.dita

  • 2-0supported-archives.dita

  • 2-0supported-docs-files.dita

Support for additional formats is being added on an on-going basis. As features are rolled out, this page will be updated.

Note

Encrypted documents/files must be decrypted before they can be isolated in the document/archive viewer and scanned by some content inspection engines. See encrypted-files-docs-policy.dita for details about creating policy for encrypted document/file download.

Note

MS Office files encrypted with ECMA-376 (Extensible Encryption) or XOR Obfuscation are not currently supported. Other limitations are noted in the tables below.
